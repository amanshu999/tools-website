# Tools Hub

**Your Daily Tools, All in One Place.**

Tools Hub is a comprehensive collection of free, easy-to-use online utilities designed to simplify your everyday tasks. Whether you need to convert an image, calculate a loan, or format text, Tools Hub has you covered.

## Features

The platform offers a variety of tools categorized for your convenience:

### 🖼️ Image Tools
- **Image Converter**: Convert images between various formats (JPG, PNG, WebP, etc.).
- **Image Compressor**: Reduce image file size without significant quality loss.
- **Image Resizer**: Change the dimensions of your images quickly.
- **Image Cropper**: Crop images to your desired aspect ratio or selection.
- **Image to Text (OCR)**: Extract text from images using Optical Character Recognition.
- **Image Rotator/Flipper**: Rotate or flip your images horizontally or vertically.

### 🧮 Calculator Tools
- **Scientific Calculator**: Advanced calculator for scientific and math functions.
- **Age Calculator**: Calculate age based on date of birth accurately.
- **BMI Calculator**: Calculate your Body Mass Index (BMI) for health insights.
- **Loan EMI Calculator**: Calculate Equated Monthly Installments for loans.
- **GST Calculator**: Calculate Goods and Services Tax amounts easily.
- **Currency Converter**: Convert between different world currencies.

### 🔤 Text Tools
- **Text Case Converter**: Convert text to uppercase, lowercase, title case, etc.
- **Word & Character Counter**: Count words, characters, sentences, and paragraphs.
- **Remove Duplicate Lines**: Easily remove duplicate lines from your text.
- **Text Sorter**: Sort lines of text alphabetically or numerically.
- **Text Reverser**: Reverse the order of characters or words in text.
- **Text Encryptor/Decryptor**: Encrypt and decrypt text messages for security.

### 🛠️ Utility Tools
- **QR Code Generator**: Create custom QR codes.
- **Barcode Generator**: Generate various types of barcodes easily.
- **Random Password Generator**: Generate strong, secure random passwords.
- **UUID Generator**: Generate Universally Unique Identifiers (UUIDs).
- **Unit Converter**: Convert between various units.
- **Timezone Converter**: Easily convert times between different time zones.

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: Custom styles ensuring a responsive and modern user interface.
- **Vanilla JavaScript**: Providing functionality for all tools without external dependencies (except for specific APIs or libraries used in individual tools).
- **FontAwesome**: Used for beautiful, scalable vector icons.
